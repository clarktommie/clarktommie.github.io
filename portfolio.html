Book Browser Dashboard

End-to-end pipeline for scraping book data, generating AI summaries, storing results in Supabase, and visualizing in a Streamlit dashboard.

USAA Fraud Detection & Storytelling Dashboard

Multimodal analytics pipeline for regulatory fraud/compliance press releases, built for DTSC 3602 at UNC Charlotte.

  • Scrapes Federal Reserve/related regulator releases (HTML + PDF) and standardizes metadata
  • Supabase as relational + vector store with SentenceTransformer (all-MiniLM-L6-v2) embeddings
  • Streamlit dashboard with semantic search, focus-term trends, and OpenAI-powered summaries
  • Library of curated semantic collections for quick retrieval and review
