Book Browser Dashboard

Book Browser Dashboard (Supabase + Streamlit + Modal)
Developed by Tommie Clark, this project demonstrates an end-to-end pipeline: scraping book data, generating AI summaries, storing results in Supabase, and exploring them in a Streamlit dashboard. The app runs locally or can be deployed to Modal for a persistent, serverless web app.

  • Scrape book details from "Books to Scrape" (title, price, availability, rating)
  • Generate concise AI summaries per book
  • Store and query structured data in Supabase (Postgres)
  • Visualize in Streamlit with filters, charts, and expandable summaries
  • Deployed as a public web app on Modal

Tech Stack: Python 3.12, Streamlit, Supabase, Pandas, Plotly, Pydeck, Folium, Modal
